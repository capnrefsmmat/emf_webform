Requirements
------------

This module currently requires Drupal 7.x, Webform 3.x, the Libraries API
module, and a copy of the createsend-php library in your libraries folder:

https://github.com/campaignmonitor/createsend-php

  1. Navigate to your 'sites/all/libraries' folder
  2. Execute 'git clone https://github.com/campaignmonitor/createsend-php.git'
